# 보상 없는 상충 목표 LLM 정렬
**부제:** Reward-free Alignment for Conflicting Objectives

## 한 줄 결론
페어와이즈 선호 데이터를 활용한 클립 기반 갈등 회피 그래디언트 하강으로 다중 상충 목표 정렬에서 우수한 파레토 트레이드오프를 달성한다.

## TL;DR (요약)
- 다중 상충 목표가 존재하는 LLM(대형 언어 모델, Large Language Model) 정렬에서 기존 가중 손실 방식은 불안정한 학습과 열악한 파레토 성능을 보인다.  
- RACO(Reward-free Alignment framework for Conflicted Objectives)는 보상 모델 없이 페어와이즈 선호 데이터를 직접 활용해 그래디언트 충돌을 해소한다.  
- 새로운 클립 변형의 갈등 회피 그래디언트 하강법으로 사용자 지정 목표 가중치를 존중하는 파레토 임계점 수렴을 이론적으로 보장한다.  
- 이항 목적 설정에서 클리핑은 수렴 속도를 엄격히 개선하며, 요약 및 안전성 정렬 실험에서 기존 다중 목표 정렬 기법 대비 우수한 트레이드오프를 보여준다.

## 문제 정의(Problem)
대형 언어 모델(LLM)은 종종 요약의 정확성, 안전성, 응답 속도 등 여러 상충하는 목표를 동시에 만족해야 하는 실제 환경에 배포된다.  
하지만 단순 가중치 합산 방식의 손실 함수는 목표 간 그래디언트 충돌을 효과적으로 해소하지 못해 불안정한 학습과 열등한 파레토 트레이드오프를 초래한다.  
기존의 다목적 최적화 기법은 보상 모델(reward model)을 명시적으로 학습해 각 목표를 수치화하지만, 이는 추가적인 복잡성을 더하고 사용자가 지정한 선호도를 왜곡할 수 있다.

## 제안 방법(Method)
RACO(Reward-free Alignment framework for Conflicted Objectives)는 보상 모델 없이 페어와이즈(pairwise) 선호 데이터를 활용한다.  
1. 페어와이즈 선호 데이터로부터 각 샘플 쌍에 대한 사용자 선호를 직접 수집하고, 이를 그래디언트 방향 결정에 반영한다.  
2. 갈등 회피(conflict-averse) 그래디언트 하강법을 도입하되, 과도한 보수적 업데이트를 피하기 위해 ‘클립 변형(clipped variant)’을 적용한다.  
   - 클리핑은 각 목표 그래디언트 벡터의 내적 등을 이용해 충돌 정도를 측정하고, 충돌이 심한 경우 업데이트 크기를 제한한다.  
3. 이론적 분석을 통해 사용자 지정 목표 가중치를 존중하며 파레토 임계점(Pareto-critical point)에 수렴함을 증명한다.  
4. 특히 두 개의 상충 목표(two-objective) 환경에서는 클리핑이 없는 방법에 비해 수렴 속도를 엄격히 개선함을 보인다.  
5. 추가적으로 다양한 휴리스틱을 도입해 실제 LLM 정렬 실험(다중 목적 요약, 안전성 정렬)에서의 성능을 향상시킨다.

## 핵심 기여/차별점(Contributions)
- 보상 모델 없이 페어와이즈 선호 데이터만으로 다중 상충 목표 정렬 문제를 직접 다루는 RACO 프레임워크 제안  
- 클립 변형 갈등 회피 그래디언트 하강법을 통한 파레토 임계점 수렴 보장 및 이항 목적 설정에서의 수렴 속도 개선 이론 제공  
- Qwen 3, Llama 3, Gemma 3 등 다양한 LLM 계열에서 다중 목적 요약 및 안전성 정렬 실험을 통해 일관되게 우수한 파레토 트레이드오프 달성

## 한계/리스크(Limitations)
- 페어와이즈 선호 데이터에 대한 의존도가 높아, 데이터 수집 비용 및 라벨링 노동이 요구된다.  
- 클리핑 기반 수렴 속도 개선은 이항(two-objective) 설정에서만 엄격히 분석되었으며, 다목적(multi-objective) 이상 설정에서의 이론적 수렴 속도 개선 여부는 초록 기준으로는 확인 불가하다.  
- 제안된 휴리스틱 세부 사항과 그 일반화 가능성은 초록만으로는 파악하기 어려우며, 실제 구현에선 추가 검증이 필요하다.

## 실무 적용 아이디어(Practical Takeaways)
- 다중 상충 목표 튜닝 시 보상 모델 없이도 페어와이즈 선호 데이터를 수집해 적용해 볼 것  
- 그래디언트 충돌이 발생하는 경우, 클립 변형 갈등 회피 기법을 도입해 파레토 성능을 개선하도록 설정  
- LLM 파인튜닝 워크플로우에 RACO 프레임워크를 통합해 요약, 챗봇 안전성, 응답 속도 등 다중 목표를 균형 있게 만족시킬 수 있는지 실험
