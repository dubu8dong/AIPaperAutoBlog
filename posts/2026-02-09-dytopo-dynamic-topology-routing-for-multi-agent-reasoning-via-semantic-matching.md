# 동적 토폴로지 기반 에이전트 협업
**부제:** DyTopo: Dynamic Topology Routing for Multi-Agent Reasoning via Semantic Matching

## 한 줄 결론
DyTopo는 라운드별 동적 통신 그래프를 구성해 에이전트 간 메시지 라우팅을 최적화하여 기존 고정 토폴로지 대비 성능과 해석 가능성을 동시에 제고한다.

## TL;DR (요약)
- 기존 멀티에이전트 시스템은 전체 추론 과정에서 고정된 통신 패턴을 사용해 단계별 요구에 비효율적이다.
- DyTopo는 매 라운드마다 매니저(Manager)의 목표를 기반으로 경량 자연어 need/offer 디스크립터를 생성하고, 의미 기반 매칭으로 희소한 방향성 그래프를 재구성한다.
- 각 에이전트는 해당 그래프 간선에 따라 사적 메시지를 주고받아 불필요한 통신을 줄인다.
- 코드 생성과 수학적 추론 벤치마크에서 강력한 비교 기법 대비 평균 6.2% 포인트 성능 향상을 보였다.
- 동적 그래프 구조를 통해 커뮤니케이션 경로 변화를 시각적으로 분석할 수 있어 디버깅 및 감사가 용이하다.

## 문제 정의(Problem)
- 다중 대화형 대형 언어 모델(Multi-Agent LLM) 기반 멀티라운드 추론 시스템은 각 라운드마다 상이한 정보 요구가 발생하지만, 대부분의 기존 파이프라인은 전체 대화 궤적(Trajectory)에 걸쳐 고정된 통신 구조를 사용한다.
- 이로 인해 단계별 필요에 맞춰 에이전트 간 메시징을 탄력적으로 조정하지 못해 통신 과부하 및 정보 누락이 발생할 수 있다.

## 제안 방법(Method)
- DyTopo 프레임워크는 매 라운드 초기 매니저가 제시한 목표(Round Goal)에 따라 각 에이전트가 ‘need’(요구)와 ‘offer’(제공) 디스크립터를 자연어 형태로 생성하도록 유도한다.
- 생성된 디스크립터를 임베딩하고 의미 유사도 기반 매칭을 수행해, 에이전트 간의 희소한 방향성 통신 그래프를 구성한다.
- 구성된 그래프의 간선에 따라 에이전트는 사적(Private) 메시지를 주고받으며, 비관련 에이전트로의 불필요한 브로드캐스트를 방지한다.
- 해당 절차를 각 라운드별로 반복해 통신 구조를 지속적으로 재구성한다.

## 핵심 기여/차별점(Contributions)
- 라운드별로 동적으로 재구성되는 희소한 방향성 통신 토폴로지를 제안하여 단계별 요구에 맞춘 메시지 라우팅을 실현  
- 경량 자연어 ‘need’과 ‘offer’ 디스크립터를 통한 의미 기반 매칭으로 불필요한 통신을 줄이고 효율성을 제고  
- 코드 생성과 수학적 추론 벤치마크에서 강력한 비교 기법 대비 평균 6.2% 포인트 성능 향상 및 해석 가능한 커뮤니케이션 트레이스 제공  

## 한계/리스크(Limitations)
- 각 라운드별 그래프 재구성 및 의미 매칭에 따른 연산 및 지연(latency) 비용이 초록에는 명시되지 않음.  
- ‘need’/‘offer’ 디스크립터 품질에 따른 매칭 정확도와 오분류 가능성은 초록 기준으로 확인 불가.  
- 에이전트 수나 복잡도 증가 시 성능 확장성 및 통신 복잡도 변화에 대한 정보는 초록에서 확인 불가.  

## 실무 적용 아이디어(Practical Takeaways)
- 대화형 AI 에이전트 협업 시스템에 DyTopo 방식을 도입해 라운드별 통신 패턴을 동적으로 조정, 불필요한 메시징을 줄여 대역폭 절감  
- need/offer 디스크립터를 각 에이전트 업무 프로토콜로 활용해 역할 기반 라우팅 정책을 간편하게 설계  
- 동적 토폴로지 변화 로그를 활용해 커뮤니케이션 경로를 시각화하고, 디버깅 및 보안 감사(감시) 목적으로 활용
