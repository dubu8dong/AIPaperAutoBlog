# DARE-bench 데이터 분석 벤치

**부제:** DARE-bench: Evaluating Modeling and Instruction Fidelity of LLMs in Data Science

## 한 줄 결론
DARE-bench는 검증 가능한 데이터 과학 태스크 기반 벤치마크로, LLM 평가와 미세조정용 학습 데이터로 활용할 수 있다.

## TL;DR (요약)
- 복잡한 다단계 데이터 과학 작업을 다루는 대형 언어 모델(LLM)의 평가 표준이 부족하다.  
- DARE-bench는 6,300개의 Kaggle 유도 태스크와 검증된 그라운드 트루스를 포함해 객관적인 평가를 지원한다.  
- 학습용 대규모 데이터와 평가용 세트를 함께 제공하여 미세조정(fine-tuning)에도 활용 가능하다.  
- 초기 실험에서 다수의 LLM이 모델링 작업에서 낮은 성능을 보였으며, 학습 데이터 활용 시 정확도가 크게 향상되었다.  
- 본 벤치마크는 평가 지표와 학습 데이터를 동시에 제공해 연구 및 실무 도입 시 유용하다.

## 문제 정의(Problem)
대형 언어 모델을 활용해 데이터 전처리, 특징 공학(feature engineering), 모델 학습 및 평가 등 일련의 데이터 과학 워크플로우를 자동화하려는 수요가 증가하고 있다.  
그러나 기존 벤치마크들은 주로 단일 입력·출력 쌍을 평가하거나 인간 평가자에 의존해 프로세스 준수(process fidelity)와 지침 충실도(instruction adherence)를 체계적으로 측정하기 어렵다.  
또한, 고품질로 라벨링된 학습 데이터가 부족해 실제 현업에 적용할 때 성능 개선을 위한 미세조정이 제한적이다.

## 제안 방법(Method)
DARE-bench는 머신러닝 모델링 및 데이터 과학 지침 수행 능력을 평가하기 위해 설계된 벤치마크다.  
6,300개의 태스크는 Kaggle 경진대회 데이터를 기반으로 생성되었으며, 각 단계별 정답(그라운드 트루스)이 명확히 검증 가능하다.  
평가 과정에서는 모델이 제시된 지침(instruction)을 얼마나 충실히 따랐는지와 올바른 중간 결과(intermediate outputs) 산출 여부를 자동으로 점검한다.

학습용 데이터와 평가용 세트를 분리 제공하며, 이를 통해 연구자는 벤치마크를 이용한 직접 평가뿐 아니라 미세조정(fine-tuning)에도 활용할 수 있다.  
논문에서는 지도학습(supervised fine-tuning)과 강화학습(reinforcement learning)을 적용해 Qwen3 계열 모델들의 정확도가 각각 1.83배, 8배 이상 향상된 사례를 제시했다.

## 핵심 기여/차별점(Contributions)
- 6,300개의 Kaggle 유도 태스크 기반 대규모 벤치마크 구축  
- 검증 가능한 그라운드 트루스 제공으로 객관적·재현 가능한 평가 지원  
- 학습용 대규모 데이터와 평가 세트를 함께 제공해 미세조정 활용 가능  

## 한계/리스크(Limitations)
- 태스크가 Kaggle 데이터에 기반해 일반화된 현업 환경 전반을 대표하지 않을 수 있다.  
- 모델 성능 향상 폭은 초록 기준으로만 제시되어, 세부 실험 조건과 재현 가능성은 초록에서 확인 불가하다.

## 실무 적용 아이디어(Practical Takeaways)
- LLM을 데이터 파이프라인 자동화 도구로 도입 전, DARE-bench로 지침 충실도와 중간 결과 정확도를 사전 평가  
- 벤치마크 학습 데이터를 활용해 도메인 특화 미세조정을 수행하고, 성능 개선 폭을 모니터링  
- 평가 지표로 instruction adherence와 process fidelity를 설정해 자동화 워크플로우 안정성 강화
