# 앵커링 기반 모델 일치

**부제:** Model Agreement via Anchoring

## 한 줄 결론
앵커링 기법을 활용해 스태킹, 부스팅, 신경망 구조 탐색, 회귀 트리 등 다양한 학습 방식에서 독립 학습된 모델 간 예측 불일치를 주요 파라미터 수 증가에 따라 0에 수렴하도록 이론적으로 보장했다.

## TL;DR (요약)
- 모델 불일치(model disagreement)는 독립 학습된 두 모델의 예측값 차이(제곱 오차 기준) 기댓값으로 정의된다.  
- 앵커링(anchoring)은 두 모델의 평균을 분석 앵커(anchor)로 삼아 불일치 상한을 도출하는 일반 기법이다.  
- 이 기법을 스태킹, 그래디언트 부스팅, 신경망 구조 탐색, 고정 깊이 회귀 트리 등 네 가지 방법에 적용해 불일치가 주요 파라미터(스택 수, 반복 수, 아키텍처 크기, 트리 깊이)에 따라 0으로 수렴함을 증명했다.  
- 초기 분석은 1차원 제곱 오차 손실에 국한되지만, 강체 convex 손실 및 다차원 회귀 문제로 일반화가 가능하다.

## 문제 정의(Problem)
머신러닝 모델 불일치(model disagreement)는 동일한 데이터 분포에서 독립적으로 학습된 두 모델이 예측에서 보이는 차이를 의미한다. 본 연구에서는 실수값 회귀 문제를 대상으로, 각 모델이 서로 다른 독립 표본을 사용해 학습된 후 두 모델 예측값의 제곱 차이의 기댓값을 불일치 척도로 삼는다. 이러한 불일치는 앙상블 성능 저하, 불확실성 측정의 왜곡, 신뢰도 보장 문제 등 실무에서 다양한 부작용을 초래할 수 있으며, 이를 제어하고 감소시킬 수 있는 원리 기반 분석 기법이 필요하다.

## 제안 방법(Method)
논문은 ‘앵커링(anchoring)’이라 명명한 일반 분석 기법을 제안한다. 두 독립 학습 모델 \(f\)와 \(g\)의 예측 불일치를 직접 다루기보다, 두 모델의 평균 \(h=(f+g)/2\)를 앵커(anchor)로 삼아
  \[
    \mathbb{E}[(f(x)-g(x))^2]
    \le \mathbb{E}[(f(x)-h(x))^2] + \mathbb{E}[(g(x)-h(x))^2]
  \]
와 같은 형태의 상한을 도출한다. 이후 이 일반 기법을 네 가지 대표 방법에 적용하여 다음과 같은 결과를 이끌어냈다.
- 스태킹(모델 앙상블) 기법: 모델 개수 \(k\)가 증가할수록 불일치가 \(O(1/k)\) 수준으로 감소.  
- 그래디언트 부스팅: 반복 횟수 \(k\)에 따라 불일치가 일정 속도로 수렴.  
- 신경망 아키텍처 검색: 아키텍처 크기 \(n\)를 확장하면 불일치가 0에 근접.  
- 고정 깊이 회귀 트리: 트리 깊이 \(d\) 증가 시 불일치가 감소.  
이 모든 분석은 1차원 제곱 오차 설정에서 시작하지만, 논문에서는 강체(convex) 손실 및 다차원 회귀 문제까지 결과를 확장했다고 주장한다.

## 핵심 기여/차별점(Contributions)
- 앵커링(anchoring)이라는 단일 분석 프레임워크를 제안해 독립 학습 모델의 불일치를 이론적으로 상한화함.  
- 스태킹, 그래디언트 부스팅, 신경망 구조 탐색, 회귀 트리 등 네 가지 주요 학습 기법에 기법을 적용해 불일치 제로 수렴을 보장.  
- 1차원 제곱 오차 손실에서 도출한 결과를 강체(convex) 손실 및 다차원 회귀로 확장 가능한 형태로 제시.

## 한계/리스크(Limitations)
- 초록 기준으로는 초기 분석이 1차원 실수 회귀 및 제곱 오차 손실에 국한됨.  
- 비강체(non-strongly convex) 손실 함수나 분류(classification) 문제로의 적용 가능성은 확인 불가.  
- 실제 복잡한 데이터 분포나 상호작용(데이터 편향, 동시 학습 상관관계 등)에 대한 고려는 이루어지지 않음.

## 실무 적용 아이디어(Practical Takeaways)
- 모델 앙상블 시 스태킹에 포함하는 모델 수를 늘려 예측 간 불일치를 감소시키고 안정성을 높일 수 있다.  
- 그래디언트 부스팅의 반복 횟수를 조정해 모델 간 일관성을 제어하고 앙상블 예측 편차를 줄일 수 있다.  
- 신경망 구조 탐색과 회귀 트리 깊이 확장을 통해 모델 간 예측 차이를 완화함으로써 앙상블 신뢰도를 향상시킬 수 있다.
