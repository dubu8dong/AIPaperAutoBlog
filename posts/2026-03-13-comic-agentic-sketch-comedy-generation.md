# AI기반 스케치 코미디 제작

**부제:** COMIC: Agentic Sketch Comedy Generation

## 한 줄 결론
대규모 에이전트 기반 프레임워크로 스케치 코미디 영상을 자동 생성한다.

## TL;DR (요약)
- AI 시스템이 캐릭터 레퍼런스에서부터 코미디 스케치 영상을 자동으로 생성  
- 제작 스튜디오 역할을 모방한 에이전트들이 반복 경쟁·평가·개선을 통해 품질과 다양성 확보  
- LLM(대형 언어 모델) 기반 크리틱을 유튜브 코미디 영상 분석으로 훈련해 유머 평가  
- 실험 결과 전문 제작물 수준에 근접하며 동영상 생성 분야 최신 성능 달성  

## 문제 정의(Problem)
오늘날 짧은 형태의 코미디 스케치는 아이디어 발상, 대본 작성, 스토리보드, 촬영·편집 등 복합적 제작 과정을 필요로 한다.  
이 과정을 사람이 아닌 AI만으로 자동화하여 SNL(Saturday Night Live)과 유사한 수준의 영상 콘텐츠를 만들기 위한 프레임워크가 부재하다.

## 제안 방법(Method)
1. 캐릭터 레퍼런스 입력을 기반으로 에이전트 집단 초기화  
2. 각 에이전트는 실제 스튜디오 역할(아이디어 제안, 스크립트 작성, 시각화 등)을 모사  
3. 반복적인 ‘경쟁–평가–개선’ 루프를 통해 아이디어와 출력물의 질과 다양성 향상  
4. 대형 언어 모델(LLM) 기반 크리틱을 구축, 유튜브 코미디 동영상 코퍼스 분석을 통해 시청자 선호에 정렬된 유머 평가  
5. 최종적으로 스케치 시나리오, 스토리보드, 영상 자막 등 단계를 종합해 자동으로 코미디 비디오 생성  

## 핵심 기여/차별점(Contributions)
- 에이전트 기반 자동화 파이프라인: 실제 제작 스튜디오 워크플로우를 에이전트로 모사  
- LLM 크리틱 도입: 시청자 선호를 반영한 유머 평가로 생성물 품질 보증  
- 스케치 코미디 영상에서 전문 제작물 수준에 근접, 동영상 생성 분야 최신 성능 달성  

## 한계/리스크(Limitations)
- 실제 시청자 평가나 문화적·지역적 맥락을 고려한 유머 적합성 평가 범위는 초록 기준으로는 확인 불가  
- 코미디 스타일 편향(bias) 문제 및 생성물의 윤리적·저작권적 리스크가 있을 수 있음  
- 시스템의 계산 자원 요구량 및 실시간 적용 가능성은 초록 기준으로는 확인 불가  

## 실무 적용 아이디어(Practical Takeaways)
- 사내 콘텐츠 제작 프로세스에 에이전트 기반 아이디어 제안 및 스크립트 보조 기능 도입  
- LLM 크리틱을 활용해 자체 영상 제작물의 유머 품질 평가 및 피드백 루프 구축  
- 자동화된 워크플로우를 통해 기획 회의와 초기 스토리보드 단계에서 시간 단축 효과 기대
