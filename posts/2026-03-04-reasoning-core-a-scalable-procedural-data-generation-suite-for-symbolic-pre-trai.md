# 확장 가능한 기호 추론 데이터

**부제:** Reasoning Core: A Scalable Procedural Data Generation Suite for Symbolic Pre-training and Post-Training

## 한 줄 결론
Reasoning Core는 PDDL(Planning Domain Definition Language), 1차 술어 논리(First-Order Logic), 문맥 자유 문법(Context-Free Grammar), 인과 추론 및 연립 방정식 등 다섯 개 핵심 기호 추론 도메인에 대해 검증 가능한 절차적 데이터 생성 및 난이도 조절 기능을 제공하며, 이를 사전학습 데이터로 활용해 언어 모델의 추론 성능을 개선한다.

## TL;DR (요약)
- 기존 절차적 데이터 생성기는 고정된 퍼즐 또는 제한된 템플릿에 의존해 분포적 다양성이 부족하다.
- Reasoning Core는 PDDL, 1차 술어 논리, 문맥 자유 문법, 무작위 베이지안 네트워크 기반 인과 추론, 연립 방정식 시스템 등 다섯 개 도메인에서 절차적 데이터를 생성하고 외부 솔버로 검증한다.
- 학습 난이도를 연속적으로 제어해 커리큘럼 설계를 지원하며, 솔버 추론 과정을 레이블로 포함해 초기 지도학습이 가능하다.
- 같은 인터페이스를 강화학습(reinforcement learning)의 보상 함수로도 활용할 수 있다.
- 데이터를 사전학습에 혼합하면 언어 모델의 추론 능력이 향상되면서 언어 모델 품질은 유지되거나 소폭 개선된다.

## 문제 정의(Problem)
- 언어 모델의 추론 능력을 확장하기 위해 기호(symbolic) 수준의 검증 가능한 데이터를 활용하는 연구가 필요하다.
- 그러나 기존 절차적 데이터 생성기는 고정된 퍼즐이나 제한된 템플릿에 의존하여 대규모로 분포적 다양성을 확보하기 어렵다.

## 제안 방법(Method)
- Reasoning Core: 다섯 개 핵심 기호 추론 도메인에 대해 절차적 데이터를 생성하는 스위트
  1) PDDL(Planning Domain Definition Language)을 이용한 무작위 계획 도메인 생성  
  2) 1차 술어 논리(First-Order Logic)와 동등성 생성  
  3) 문맥 자유 문법(Context-Free Grammar) 기반 파싱 및 생성  
  4) 무작위 베이지안 네트워크 기반 인과 구조 생성  
  5) 연립 방정식 시스템  
- 각 예제는 외부 솔버를 통해 엄밀하게 검증  
- 학습 난이도를 연속적으로 제어해 커리큘럼 설계를 지원  
- 솔버 유도 추론(trace)을 포함해 초기 단계의 지도학습에 활용  
- 동일 인터페이스를 강화학습 보상 함수로 제공  

## 핵심 기여/차별점(Contributions)
- 절차적 생성과 외부 솔버 검증을 결합해 다섯 개 기호 추론 도메인의 대규모 데이터셋을 확장 가능하게 제공  
- 연속 난이도 제어 및 솔버 추론 과정을 레이블로 활용해 초기 사전학습부터 지도학습 및 강화학습까지 통합 지원  
- 생성 데이터 혼합 시 언어 모델의 추론 성능을 향상시키면서 언어 모델 품질을 유지 또는 소폭 개선함을 실험적으로 검증  

## 한계/리스크(Limitations)
- 실제 대규모 언어 모델 사전학습에 필요한 컴퓨팅 비용 및 데이터 생성 오버헤드는 초록 기준으로 확인 불가  
- 다섯 개 도메인 외 추가 기호 추론 분야로의 확장 가능성과 일반화 범위는 초록 기준으로 확인 불가  

## 실무 적용 아이디어(Practical Takeaways)
- Reasoning Core를 사전학습 데이터 파이프라인에 통합해 언어 모델의 기호 추론 능력 강화  
- 솔버 유도 추론을 레이블로 활용한 지도학습으로 초기 단계에서 구체적 추론 패턴 학습 유도  
- 강화학습 보상 함수로 적용해 언어 모델이 복합 기호 추론 문제를 직접 탐색하고 최적화하도록 지원
